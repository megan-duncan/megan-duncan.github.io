Margin, Border or Padding.

Margin border and padding are all a part of the CSS box model, so first, I’ll go into a little about that.

The Box Model

To the left is a diagram of a box model. As you can see, it is made up of four layers

  • Content Box - the innermost layer where your content is displayed.
  • Padding Box - this sits right around your content box, inside the border and is white space.
  • Border Box - this wraps around your content and padding.
  • Margin Box - the outermost layer, wrapping your element and creating whitespace between it and any other elements.

Each of these layers can be altered to change the way your element interacts with other elements on the page, below I will go into more detail about what these layers are, and how to alter them. But first, here is a handy analogy I came across online may help you to understand the box model.

Imagine that you have a set of framed and matted photographs hanging on a wall.

Each photo is surrounded by a mat that puts space between the photo and the frame. That space would be the padding.

The frame encloses the mat and the image. That would be the border.

Each picture has a set amount of space around the outside. That is the margin. If two pictures are handing side by side with four inches between them, then we might say that they each have a two inch margin. The margin is the distance between an element and something else.

Content

Your content is the text/image (or anything else, really) that you want to be displayed on your page. You can change the size of your content using properties like width and height

Padding

The padding is the whitespace between the content and the border.

You can change the size of the padding by using the property padding followed by the size you want.

If you wanted to change the size of each side individually, you would use the following properties: padding-top, padding-bottom, padding-left, padding-right.

The shorthand code for this would be - padding: 20px;

The code for this would be - padding-left: 40px; padding-right: 30px; padding-top: 10px; padding-bottom: 20px;

If you are using the standard box model, when you increase the size of your padding, you are increasing the size of your element.

Border

The border is the line between the padding and margin of a box. The default border is not visible on a page, but we can change the width, style and colour of the border to make it more of a feature.

Like with the padding, you can opt to change all 4 borders at once with the border property or each side individually using border-top etc followed by what you are wanting to change eg border topwidth/style/color

Margin

The margin is the white space around your box, when you increase its size, you push other elements further away from the box.

We can change the size of all 4 margins using the property margin or each individually using margin-top etc..

Margin is the only property in the box that can have a negative value, this will cause it to overlap the other things on the page.

One thing to be aware of, is that positive margins don’t overlap. This means 2 elements side by side with margin of 20px will create a space of 40px between them.

The size of the margin is always added on top of the size of the box.

The code for this would be - margin-left: 40px; margin-right: 30px; margin-top: 10px; margin-bottom: 20px;

These two elements with 20px margins create a 40px space when next to each other.
