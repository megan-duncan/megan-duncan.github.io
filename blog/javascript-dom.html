The DOM

The DOM stands for “Document Object Model” and is a programming interface for webpages.

It represents the page in a tree-like structure, to make it easier for other programs to add/change or delete elements from the page.

You may use the DOM to find all instances of an 'h1' element in a webpage, so you can add or alter its class.

To do this, you can use the document method querySelectorAll("h1"); this will return a list of all the elements on the page that have a 'h1' tag.

Arrays and Objects

Arrays store data in list form, using a numbered index, starting at [0].

Objects store data as properties, using a named index

You access data from an array using its index number, using square bracket notation, whereas you access data from an object using dot notation.
